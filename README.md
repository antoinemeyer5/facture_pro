## Fonctionnalités

- Champs éditables
- Plusieurs types de champs : date, nombre, texte
- Ajout de tâches réalisées à la facture
- Export de la facture en PDF

## Developer's zone

### Launch

- `npm install`
- `npm run dev` or `npm run dev -- --open`

### Build

- `npm run build`
- `npm run preview`

### Technical stack

- SvelteKit 2.22.0
